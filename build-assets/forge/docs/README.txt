GoForge — the Forge · static docs (canonical copy)
===================================================

`index.html` in this directory is the **canonical** Alfred Linux / GoForge help
page for `/forge/docs/` on alfredlinux.com.

Deploy to the live site (paths may require root or apache ownership on your host):

  install -m0644 build-assets/forge/docs/index.html \
    /path/to/alfredlinux.com/public_html/forge/docs/index.html

Or `sudo cp` after review. Keep in sync when you change forge UX copy.
