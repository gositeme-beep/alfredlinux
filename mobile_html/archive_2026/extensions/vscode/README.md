# Alfred AI — VS Code Extension

AI-powered coding assistant with 1,290+ tools, right in your editor.

## Features

- **Chat Panel** — Ask Alfred anything from the sidebar (Ctrl+Shift+A)
- **Code Explanation** — Select code → right-click → "Alfred: Explain"
- **Refactoring** — AI-powered code refactoring suggestions
- **Test Generation** — Auto-generate unit tests for selected code
- **Documentation** — Generate JSDoc/docstrings in one click
- **Code Review** — Security, bugs, and improvement analysis
- **Error Fixing** — Auto-detect and fix diagnostics errors
- **Optimization** — Performance optimization suggestions

## Quick Start

1. Install the extension
2. Set your API key: Settings → Alfred AI → API Key
3. Press `Ctrl+Shift+A` to open the chat panel
4. Select code and right-click for context menu actions

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl+Shift+A` | Open Alfred Chat |
| `Ctrl+Shift+Q` | Quick question |

## Right-Click Menu

Select any code and right-click to see Alfred options:
- Explain Selected Code
- Refactor Selected Code
- Generate Tests
- Generate Documentation
- Code Review
- Optimize Code

## Configuration

| Setting | Default | Description |
|---------|---------|-------------|
| `alfred.apiKey` | — | Your API key |
| `alfred.apiUrl` | `https://gositeme.com/api/v1` | API endpoint |
| `alfred.model` | `auto` | Model tier: auto, fast, balanced, powerful |
