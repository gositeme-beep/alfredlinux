
# Chess Sound Effects

This folder contains sound effects for the chess game:

## Available Sounds:

- `chess-move.mp3` - Standard piece movement sound
- `chess-capture.mp3` - Sound played when capturing a piece
- `chess-check.mp3` - Alert sound when a king is in check
- `chess-win.mp3` - Victory fanfare sound
- `chess-ambient.mp3` - Optional background ambient music for game atmosphere

## Usage:

These sounds are loaded dynamically in the BoardArea and ChessBoard components.
