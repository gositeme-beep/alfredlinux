
# Chess Pieces Image Assets

This folder contains the image assets for the chess pieces in different styles:

## Available Sets:

1. **realistic** - 3D rendered realistic chess pieces
2. **marble** - Elegant marble chess pieces
3. **metal** - Metallic chess pieces

## File Naming Convention:

Each piece follows this naming pattern: `[color]_[piece-type].png`

For example:
- `white_king.png`
- `black_queen.png`
- `white_pawn.png`

## Notes:

The images should be transparent PNGs with good resolution for sharp display on high-DPI devices.
