Omahon seal — not locked to one file
=====================================
The “heraldic” / earlier picks were **interim** only. For GA, you generate what you
want, then run **immediately before** the ISO build:

  bash build-assets/wallpapers/scripts/apply-new-omahon-source.sh /path/to/final.png

See **../BEFORE-GA-SEAL.txt**. Variants in this folder are optional references only
until you install a new master via that script.
