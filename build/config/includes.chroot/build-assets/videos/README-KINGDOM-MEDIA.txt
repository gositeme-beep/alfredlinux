Kingdom media — PRIMARY way to close the ~1 GiB gap to ~7.77 GiB (binary)
========================================================================
This is the **main, glorious** payload for the size gate: **cinematic video + 4K/8K**
stills (worshipful art on disk for **Yeshua / Jesus** — honest bytes, not padding).

For **real** biblical story video (generative or commissioned), read
**README-PREMIUM-BIBLICAL-VIDEO.txt**. Ignore `../build-kingdom-video.sh|.py` for

This tree is copied into the live system by **section 7** of
`config/hooks/live/0285-alfred-eternal-storage.hook.chroot` →
`/usr/share/alfred-linux/kingdom-media/`.
